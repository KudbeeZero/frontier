# Development Phases

## Phase 1: Survival (Tutorial)

**Goal:** Introduce core mechanics and establish the threat.

Story beats:
- Ship systems damaged — limited weapons/maneuverability
- First contact with enemy forces
- Emergency repair sequence

Gameplay milestones:
- [ ] Basic orbital movement
- [ ] Pulse cannon operational
- [ ] First enemy wave (scouts only)
- [ ] Emergency repair collected

---

## Phase 2: Stabilization

**Goal:** Player regains full ship control.

Story beats:
- Systems come back online
- Identify enemy command structure
- Secure orbital perimeter

Gameplay milestones:
- [ ] Rail gun unlocked
- [ ] Mixed enemy waves (scouts + cruisers)
- [ ] Resource collection system active
- [ ] First upgrade purchased

---

## Phase 3: Discovery

**Goal:** Uncover the source of the invasion.

Story beats:
- Intercept enemy transmissions
- Locate forward base
- Recon mission

Gameplay milestones:
- [ ] Missile system unlocked
- [ ] Boss enemy introduced (cruiser commander)
- [ ] Intel items collectible

---

## Phase 4: Escalation

**Goal:** The enemy intensifies — player must adapt.

Story beats:
- Enemy reinforcements arrive
- Dreadnought class ships deploy
- Earth defense grid fails

Gameplay milestones:
- [ ] Dreadnought enemies active
- [ ] Drone swarm waves
- [ ] Advanced upgrades tier unlocked
- [ ] Earth health mechanic introduced

---

## Phase 5: Breakthrough

**Goal:** Strike at the enemy command ship.

Story beats:
- Locate enemy mothership
- Plan assault
- Execute strike

Gameplay milestones:
- [ ] Mothership boss fight
- [ ] Special ability unlocked
- [ ] Maximum upgrade tier

---

## Phase 6: Resolution

**Goal:** Final battle and ending.

Story beats:
- Final confrontation
- Player choice: destroy or capture
- Epilogue

Gameplay milestones:
- [ ] Final boss encounter
- [ ] Ending cutscene
- [ ] Score/leaderboard submission to ICP backend
- [ ] Achievement system

---

## Technical Milestones Per Phase

Each phase includes:
- New 3D assets or effects
- New story event data in `story-data/chapters/`
- A new canister function (Phase 6 only: on-chain score)
